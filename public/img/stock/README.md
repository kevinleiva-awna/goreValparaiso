# Fotos editoriales para la web pública

Si querés agregar fotos reales en lugar de las ilustraciones SVG actuales,
pon los archivos acá con estos nombres y dimensiones recomendadas:

## hero-valparaiso.jpg (o .webp)
- Foto panorámica de Valparaíso (cerros, puerto, paisaje regional)
- Dimensiones: 1600×900 mínimo (16:9)
- Peso recomendado: <300 KB optimizado
- Reemplazará el SVG skyline en el hero del home
- Se conecta automáticamente si existe el archivo

## consulta-default.jpg
- Foto genérica para cards de consulta sin imagen propia
- Dimensiones: 800×450 (16:9)

## como-funciona-hero.jpg
- Foto editorial para la sección "Cómo participar"
- Dimensiones: 1200×675

## faq-hero.jpg
- Foto para sección de preguntas frecuentes
- Dimensiones: 1200×675

Si no existe un archivo, el sitio sigue funcionando con la ilustración
SVG inline como fallback.
